# Human Behavior Recognition in Video Streams

Project for the course **Problem Workshop in Software Engineering**.

## Description
The goal of this project is to develop a system capable of recognizing human actions in video streams.  
The system will analyze sequences of video frames to detect behaviors such as walking, running, falling, or aggressive actions.

**Dataset:** The project uses the [PIP 370k (Stabilized)](https://github.com/visym/collector/tree/master/docs/pip_370k_stabilized) dataset for training and validation.

## Technology
The project is implemented in **Python**.
